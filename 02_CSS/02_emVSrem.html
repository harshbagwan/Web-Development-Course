EM vs REM Demo

This parent has a font size of 20px.

This child uses 2em — it becomes 40px (2 × 20).

This child uses 2rem — it becomes 32px (2 × 16, root size).

This parent has a font size of 30px.

This child uses 2em — it becomes 60px (2 × 30).

This child uses 2rem — still 32px (2 × 16, root size doesn't change).
